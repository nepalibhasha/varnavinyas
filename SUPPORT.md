# Support

## Where to ask for help

- Usage questions: open a GitHub issue using the most relevant template.
- Linguistic correctness questions: use the `Linguistic issue` template.
- Bugs: use the `Bug report` template.
- Feature ideas: use the `Feature request` template.

## Before opening an issue

- Search existing issues first.
- Include minimal reproducible input where possible.
- Include expected vs actual behavior.

## Scope

This repository is focused on Nepali orthography tooling and related
language-processing components in this workspace.

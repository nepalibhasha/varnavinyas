# varnavinyas-shabda

**Word Analysis and Morphology.**

Classifies words by origin (Tatsam/Tadbhav) and analyzes morphological structure.

## Features
- **Origin Detection**: Classifies words as Tatsam (Sanskrit-origin), Tadbhav, Deshaj, or Aagantuk.
- **Morphology**: Decomposes words into Root + Affixes.

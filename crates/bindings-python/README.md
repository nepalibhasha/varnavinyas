# varnavinyas-bindings-python

**Python Bindings for Varnavinyas.**

Exposes core functionality (spell checking, transliteration, analysis) to Python via PyO3.

## Installation

```bash
pip install varnavinyas
```

## Usage

```python
import varnavinyas
checker = varnavinyas.Parikshak()
```

# varnavinyas-sandhi

**Sound Change Rules (Sandhi).**

Implements rules for phonological changes when morphemes combine.

## Features
- **Conjunct Formation**: Rules for combining consonants (e.g., क् + ष = क्ष).
- **Vowel Sandhi**: Joining rules for vowels.
- **Splitting**: Decomposing sandhi-joined words (experimental).

## Status
✅ Stable (Core rules)

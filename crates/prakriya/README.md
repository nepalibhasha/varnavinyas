# varnavinyas-prakriya

**Derivation Engine & Rule Tracing.**

The core logic engine that explains *why* a spelling is correct or incorrect based on the Nepal Academy Orthography Standard.

## Features
- **Rule Application**: Applies Hrasva/Dirgha categories, Chandrabindu rules, etc.
- **Tracing**: Produces step-by-step derivation history with citations.
- **Standards Aligned**: Every rule maps to a specific section of the Academy standard.
